EPICS MEDM 编译

1. 下载 MEDM 源码

从 EPICS 主页下载 tar 包，解压到 <extensions>/src/。

wget https://epics.anl.gov/download/extensions/MEDM3_1_11.tar.gz
tar -vxzf MEDM3_1_13.tar.gz -C <extensions>/src/

2. 编译

make

fatal error: X11/Intrinsic.h: No such file or directory -- 需要安装 libXt
fatal error: Xm/XmP.h: No such file or directory -- 需要安装motif-dev

3. 测试

cd <extensions>/linux-86_64/
./medm
