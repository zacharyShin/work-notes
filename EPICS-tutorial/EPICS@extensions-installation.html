EPICS extensions 编译

1. 下载EPICS extensions 源码

可以从EPICS 主页下载 EPICS extensions 源码的 tar 包，解压后进行编译。

wget https://epics.anl.gov/download/extensions/extensionsTop_20120904.tar.gz
tar -vxzf extensionsTop_20120904.tar.gz -C .

或者从 Github 克隆项目到本地。

git clone https://github.com/epics-extensions/extensions.git

2. 建立 extensions 的软链接（可选）

建立软链接主要为方便之后的操作，无须记住 extensions 的版本号

ln -s extensions_20120904 extensions

3. 根据 extensions 文件夹位置修改 configure/RELEASE 文件

需要修改 configure/RELEASE 文件，设置 EPICS base 的位置

cd extensions
vi configure/RELEASE

修改 EPICS_BASE 变量为本地 EPICS base 的安装位置

4. 编译 MSI

MSI (Macro Substitution and Include Tool) 是一种通用宏替换/包含工具。有一些 EPICS 应用模块的编译要求 MSI 的安装。其源代码 tar 包同样可以从 EPICS 主页下载，解压后编译即可。

wget https://epics.anl.gov/download/extensions/msi1-7.tar.gz
tar -vxzf msi1-7.tar.gz -C .
ln -s msi1-7 msi
cd msi
make
