使用 xxx 模块测试 synApps

synApps 模块无错误编译后，可以使用 xxx 模块作为示例或模板构建一个用户目录以支持IOC。

1. 复制 xxx 模块

清理并卸载 xxx 模块，打包复制到目标位置。（以 1w1a 为例）

cd $(SYNAPPS)
mkdir ioc/
cd ioc/
mkdir 1w1a/

cd $(SYNAPPS)/support/xxx
make clean uninstall
tar cvf ../xxx.tar *

tar -xf $(SYNAPPS)/support/xxx.tar
$(SYNAPPS)/support/util/changePrefix xxx 1w1a

changePrefix 是 synApps 提供的修改前缀的工具。

2. 修改 st.Linux.<arch> 以及相关的 iocsh 文件。

3. make

4. 运行

cd iocBoot/ioc1w1a/
../../bin/linux-x86_64/1w1a st.cmd.Linux
